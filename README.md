# Создать простой AI агент на Python с использованием LangChain

## Requirements
- [high] Подключение к LM Studio: Настроить клиент LangChain для работы с локальным LM Studio по адресу http://localhost:1234/v1, включая корректную аутентификацию и проверку доступности модели.
- [high] Добавление зависимости langchain-openai: Обновить файл зависимостей (requirements.txt или pyproject.toml) и установить пакет langchain-openai, чтобы обеспечить совместимость с OpenAI‑совместимыми моделями.
- [high] Обновление кода с использованием create_agent: Переписать существующий код агента, используя современный API LangChain (create_agent), включая определение цепочки действий, инструментов и обработку запросов.
