# Создать простой AI агент на Python с использованием LangChain

## Requirements
- [high] Добавить файл requirements.txt: Создать файл requirements.txt в корне репозитория, включив в него все зависимости, необходимые для работы агента (langchain, openai, python-dotenv и т.д.).
- [high] Реализовать простого AI‑агента: Написать скрипт (например, agent.py), который инициализирует LangChain Agent, подключает LLM (OpenAI, GPT‑4 и т.п.) и умеет обрабатывать простые запросы пользователя, возвращая ответы.
- [normal] Проверить работоспособность: Запустить скрипт локально, убедиться, что агент корректно отвечает на тестовые запросы, и зафиксировать пример использования в README.
